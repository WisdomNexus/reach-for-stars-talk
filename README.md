## 在线访问


## 部署步骤(GitHub Pages,5 分钟搞定)

1. 登录 [GitHub](https://github.com),点右上角 `+` → **New repository**
2. Repository name 填:`reach-for-stars-talk`
3. 勾选 **Public**(私有库需付费才能开 Pages),点 **Create repository**
4. 进入空仓库页面 → 点 **uploading an existing file** → 把本目录下的 `index.html` 和 `README.md` 拖进去 → **Commit changes**
5. 仓库顶部 **Settings** → 左侧 **Pages** → **Source** 选 `Deploy from a branch` → **Branch** 选 `main` / `(root)` → **Save**
6. 等 30~60 秒,刷新 Pages 页面,顶部出现绿色 `Your site is live at https://xxx.github.io/reach-for-stars-talk/` 即部署成功

> 链接长期有效,免费、无流量限制,可直接发给学生/同事访问。
